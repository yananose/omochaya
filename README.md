# Omochaya Unity Libraries

Unity向けの自作ライブラリ群を管理するリポジトリです。

## 📦 収録ライブラリ (Packages)

| ライブラリ名 | 概要 | リンク |
| :--- | :--- | :--- |
| **Story** | コルーチンを置き換えるゼロアロケーション非同期タスクエンジン | [READMEはこちら](./Omochaya/Story/README.md) |
| *(FutureLibrary)* | 開発予定の新しいライブラリ | 準備中 |

## 🚀 インストール方法

使いたいライブラリのフォルダ（例: `Omochaya/Story`）をご自身のUnityプロジェクトの `Assets` フォルダ配下にコピーしてください。
